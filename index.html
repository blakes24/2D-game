Haunted Run

How to play

Collect as many potions as you can in the haunted graveyard, but watch out for the ghosts!

Use the right and left arrows to move, and the up arrow or space bar to jump.

About

This game was created for Mintbean's August hiring hackathon. The challenge was to create a 2D game using JavaScript. This project was built with JavaScript, Phaser 3, HTML, and CSS.
